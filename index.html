MemeEconomy Market Evaluator

MEME, (or MemeEconomy Market Evaluator) offers a set of tools to analyze the market at r/MemeEconomy. The project is currently in beta and frequent changes should be expected.

What is MemeEconomy?

r/MemeEconomy is a place where individuals can buy, sell, share, make, and invest in memes freely. Individuals will also get updates on the market, and be able to collaborate with other fellow meme traders.

How does it work?

Users start with 1000 MemeCoins and can freely invest in memes. Once a meme has been invested in, a bot evaluates the user's investment after 4 hours. The user will either profit or lose MemeCoins based on how many upvotes the meme recieved. For more details regarding account setup and firm transacations, visit the official site.

Available tools

Currently, MEME offers the investment return and the break-even calculators. Some tools like the Investment Analyst AI and the Portfolio Manager are still a work-in-progress and will be released in the near future.

Who's behind this?

MemeEconomy is a large community with many developers and moderators. MEME is a project by Param Thakkar and is not associated with MemeEconomy.
